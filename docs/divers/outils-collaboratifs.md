# Outils & portails collaboratifs

Cette page recense les outils et portails collaboratifs couramment utilisés dans le cadres des activités des membres du département MEE. Ces outils sont maintenus en interne par la Direction de l’Infrastructure et du Système d’Information (DISI) d’IMT Atlantique ou en externe par d’autres entités (IMT, CNRS, etc…). L’accès s’effectue généralement par authentification Campus.

## Partages

Partage (IMT Nord Europe), quota de 10Go : [https://partage.imt.fr/](https://partage.imt.fr/) (contacter le support : support-nextcloud@imt-nord-europe.fr)

## Échanges de fichier

échanger de manière sécurisé des fichiers volumineux : [https://filesender.renater.fr/](https://filesender.renater.fr/)

## Visioconférences

Webex (IMT Atlantique) : [https://imt-atlantique.webex.com/](https://imt-atlantique.webex.com/)

## Sondages

Si vous voulez créer un sondage, il est possible d’utiliser l’outil LimeSurvey dont une instance est installé sur les serveurs de l’école :

- créer un sondage : [https://hub.imt-atlantique.fr/limesurvey/admin/](https://hub.imt-atlantique.fr/limesurvey/admin/)
- documentation : [https://hub.imt-atlantique.fr/limesurvey/topo-limesurvey.pdf](https://hub.imt-atlantique.fr/limesurvey/topo-limesurvey.pdf)
