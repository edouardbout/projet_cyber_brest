# Slurm default Values

Defaults values used by `srun`, `salloc` or `sbatch`.

| Component     | Default value used                 |
| ------------- | ---------------------------------- |
| **Partition** | The default partition is `Mee`     |
| **Account**   | Your SLURM `DefaultAccount`        |
| **QoS**       | As defined by your association     |
| **CPU(s)**    | **Only 1 CPU**                     |
| **Memory**    | Minimum allocation (about 2.5GB)   |
| **Time**      | No limit for now                   |
| **Node(s)**   | 1 node                             |
| **GPU(s)**    | None (unless explicitly requested) |
